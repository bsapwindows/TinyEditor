# TinyEditor

## Usage

Paste the following code into your browser address bar:

    data:text/html,<body oninput="i.srcdoc=h.value+'<style>'+c.value+'</style><script>'+j.value+'</script>'"><style>body *{width:33.33%;height:50%}*{margin:0}#i{width:100%;border:0}</style><textarea placeholder=HTML id=h></textarea><textarea placeholder=CSS id=c></textarea><textarea placeholder=JS id=j></textarea><iframe id=i>

If you want to read the code, check out [index.html](https://github.com/umpox/TinyEditor/blob/master/index.html)
